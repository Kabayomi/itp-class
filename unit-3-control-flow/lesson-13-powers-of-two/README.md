# Powers of two

Define a function `powers_of_two` that receives a power and uses a for loop
to calculate and return 2 to that power. Do not use the math.pow or or the x**y 
operator for this assignment.


Examples:

```python
>>> powers_of_two(0)
1
>>> powers_of_two(1)
2
>>> powers_of_two(2)
4
>>> powers_of_two(3)
8
```
